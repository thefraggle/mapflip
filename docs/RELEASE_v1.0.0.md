# 🗺️ MapFlip v1.0.0

Automatically open Apple Maps links in Google Maps. Set up once, then it works invisibly.

## Features

- Automatic redirect – set up once, invisible from then on
- Supports searches, coordinates, addresses & navigation
- English & German
- No data collection, no ads
- Completely free & open source

## Install

1. Download `MapFlip-v1.0.0-debug.apk` below
2. Open the APK on your Android device
3. Follow the setup instructions in the app

> **Note:** You may need to allow installation from unknown sources.
